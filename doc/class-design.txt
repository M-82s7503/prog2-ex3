【Mapクラス】
フィールド変数：
    int start_x,start_y:スタート位置
    int mapSize:迷路サイズ
    Char map[][]:マップ(char型二次元配列)
    String goal:ゴール位置   //int?

コンストラクタ：
    引数：マップファイル
    処理：
        マップファイルを開く
        スタート位置を読み込む
        迷路サイズを読み込む
        迷路サイズの二次元配列を確保
        二次元配列に迷路を入れる
        Gを読み込んでgoalに代入
メソッド：
    printMap(Char map[][],int place):現在マップの出力


【Playerクラス】
フィールド変数：
    int place:プレイヤーの位置
    int count:歩数

コンストラクタ：
    引数：スタート位置
    処理：
        This.placeにスタート位置を代入
        This.count を0で初期化
